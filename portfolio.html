Jet Fighter Game

Jet Fighter was a project that I built for an assignment in college using the Phaser game engine. The Capstone class was meant to show how to go through the software design steps from planning all the way to marketing. This was initially a group project but ended up being written primarily by myself. The program was also the first time I had written in Typescript. Though the project is still a work in progress, it is an example of some of the work I've done.
